## 🚀 **Quick Start**

### **1. Install Dependencies**
```bash
npm install
```

### **2. Start Development Servers**
```bash
# Start API server
cd apps/api
npm run dev

# Start Web server (in new terminal)
cd apps/web
npm run dev
```

### **3. Run Tests and Validation**
```bash
# Run all validations
npm run validate:all

# Run comprehensive tests
npm run test:app

# Run everything
npm run test:all
```
