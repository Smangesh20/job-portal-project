❓ TROUBLESHOOTING:

Q: Signup not working?
A: 1. Check if server is running (npm run dev)
   2. Open DevTools console (F12)
   3. Look for OTP: 🔐 Development OTP: 123456
   4. Check Network tab for API calls

Q: Can't see OTP?
A: Open browser console (F12) → Console tab
   OTP is logged there in development mode

Q: Network error?
A: Restart development server:
   Ctrl+C → npm run dev

Q: Google OAuth not working?
A: Requires Google Cloud setup (optional)
   See SETUP.md for instructions
